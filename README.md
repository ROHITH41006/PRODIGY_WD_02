# Employee Management System

A modern Employee Management System built using React, React Router, Context API, and Local Storage.

## Features

### Authentication
- Admin Login
- Protected Routes
- Session Persistence
- Logout Functionality

### Dashboard
- Total Employees
- Total Departments
- Highest Salary
- Recent Employees

### Employee Management
- Add Employee
- View Employee List
- Edit Employee Details
- Delete Employee
- Delete Confirmation Popup

### Validation
- Required Field Validation
- Salary Validation

### Navigation
- Responsive Navbar
- Back Navigation Buttons

### Data Persistence
- Context API
- Local Storage

## Tech Stack

- React
- React Router DOM
- Context API
- Vite
- Tailwind CSS
- Local Storage

src/
